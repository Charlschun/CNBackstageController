CNBackstageDelegate Protocol Reference

Conforms to NSObject
Declared in CNBackstageDelegate.h

Overview

The CNBackstage Delegate provides a set of methods the user can implement to control the functionality of its own backstage instance. On each delegate call a related notification will be send too. So you can decide by yourself either to use the delegate mechanism or NSNotificationCenter (or both).

backstageController:willCollapseOnScreen:onToggleEdge:

Informs the delegate that the screen toggleScreen will close on edge toggleEdge.

- (void)backstageController:(CNBackstageController *)backstageController willCollapseOnScreen:(NSScreen *)toggleScreen onToggleEdge:(CNToggleEdge)toggleEdge

Parameters

toggleScreen

The screen that will close the CNBackstageController’s view.

toggleEdge

The edge the CNBackstageController’s view did appear.

Discussion

This delegate also post a CNBackstageControllerWillCloseNotification notification to the NSNotificationCenter. It sends the toggleScreen parameter as an item of the userInfo dictionary with the key toggleScreen.
